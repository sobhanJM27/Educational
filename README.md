## 🌟 Overview

Education is designed as a modern learning experience where students can discover content, follow educational paths, and interact with a fast and responsive interface.

The application focuses on:

* 📚 Course exploration
* 📝 Educational content
* ⚡ Fast page navigation
* 🎯 Responsive user experience
* 🔍 Searchable content structure
* 📱 Mobile-friendly design

---

## ✨ Core Capabilities

| Feature                   | Description                           |
| ------------------------- | ------------------------------------- |
| 📖 Educational Content    | Browse and explore learning materials |
| 🔎 Smart Navigation       | Fast routing with React Router        |
| ⚡ Server State Management | Powered by TanStack Query             |
| 🧠 Global State           | Redux Toolkit                         |
| 🎨 Modern UI              | TailwindCSS + Radix UI                |
| 📱 Responsive Design      | Optimized for all screen sizes        |
| 🔔 User Feedback          | Toast notifications                   |

## 🏗 Architecture Philosophy

This project follows a scalable frontend architecture:

```text
Pages
  │
  ▼
Components
  │
  ▼
Custom Hooks
  │
  ▼
API Layer
  │
  ▼
Server State (React Query)
```

The goal is to keep concerns separated, improve maintainability, and make future scaling easier.
